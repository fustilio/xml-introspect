## 🎯 Quick Start

```bash
# Install
npm install xml-introspect

# Sample large XML file
xml-introspect sample input.xml output.xml

# Generate XSD schema
xml-introspect schema data.xml schema.xsd

# Process from URL
xml-introspect sample https://example.com/data.xml.gz output.xml
```
